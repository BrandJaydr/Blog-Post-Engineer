Project Objectives
Phase 1 — Unblock Design Work (Today, 2 Hours)
Get category post filtering working on the existing BDP installation so page design work can proceed without interruption. No renaming, no refactoring, no GG hooks. Additive changes only.
Phase 2 — GG Blogging Engine (Ongoing)
Fork the plugin into a standalone GG ecosystem module with extended layouts, custom post type support, a cleaner admin UI, and optional GG core registration. Developed incrementally alongside other ecosystem work.

PHASE 1 — Category Filter Sprint
Objective
Add Ajax-powered category filter buttons to [bdp_post] (grid layout). When a user clicks a category button, the post grid reloads without a page refresh showing only posts from that category. All existing shortcode params remain unchanged. This directly unblocks the page layout work.

Step 2: Render filter buttons — still in bdp-post.php
Immediately before the ob_start() call that begins the output buffer, insert:
ob_start();
if ( $show_filter === 'true' && ! empty( $filter_cats ) ) {
    $conf = json_encode( array(
        'design'       => $design,
        'grid'         => $grid,
        'media_size'   => $media_size,
        'limit'        => $posts_per_page,
        'words_limit'  => $words_limit,
        'showAuthor'   => $showAuthor,
        'showDate'     => $showDate,
        'showCategory' => $showCategory,
        'showContent'  => $showContent,
        'show_tags'    => $show_tags,
        'show_comments'=> $show_comments,
        'showreadmore' => $showreadmore,
        'order'        => $order,
        'orderby'      => $orderby,
    ) );
    echo '<div class="bdp-filter-bar" data-conf='' . esc_attr( $conf ) . ''>';
    echo '<button class="bdp-filter-btn active" data-cat="">All</button>';
    foreach ( $filter_cats as $cat_ref ) {
        $term = is_numeric($cat_ref) ? get_term( (int)$cat_ref, 'category' ) : get_term_by('slug', $cat_ref, 'category');
        if ( $term && ! is_wp_error($term) ) {
            echo '<button class="bdp-filter-btn" data-cat="' . esc_attr($term->slug) . '">' . esc_html($term->name) . '</button>';
        }
    }
    echo '</div>';
}
Step 3: includes/bdp-ajax-functions.php — Ajax handler
Inside the Wpbdp_Public class constructor, add:
add_action( 'wp_ajax_bdp_filter_posts',        array($this, 'bdp_filter_posts') );
add_action( 'wp_ajax_nopriv_bdp_filter_posts', array($this, 'bdp_filter_posts') );
Then add the handler method:
function bdp_filter_posts() {
    check_ajax_referer( 'bdp_filter_nonce', 'nonce' );
    global $post;
    $atts     = ! empty( $_POST['conf'] ) ? $_POST['conf'] : array();
    $cat_slug = sanitize_text_field( $_POST['cat'] ?? '' );

    $design        = isset($atts['design']) ? sanitize_text_field($atts['design']) : 'design-1';
    $grid          = isset($atts['grid']) ? intval($atts['grid']) : 2;
    $posts_per_page= isset($atts['limit']) ? intval($atts['limit']) : 20;
    $media_size    = isset($atts['media_size']) ? sanitize_text_field($atts['media_size']) : 'large';
    $words_limit   = isset($atts['words_limit']) ? intval($atts['words_limit']) : 20;
    $showAuthor    = isset($atts['showAuthor']) && $atts['showAuthor'] === 'false' ? 'false' : 'true';
    $showDate      = isset($atts['showDate']) && $atts['showDate'] === 'false' ? 'false' : 'true';
    $showCategory  = isset($atts['showCategory']) && $atts['showCategory'] === 'false' ? 'false' : 'true';
    $showContent   = isset($atts['showContent']) && $atts['showContent'] === 'true' ? 'true' : 'false';
    $show_tags     = isset($atts['show_tags']) && $atts['show_tags'] === 'false' ? 'false' : 'true';
    $show_comments = isset($atts['show_comments']) && $atts['show_comments'] === 'false' ? 'false' : 'true';
    $showreadmore  = isset($atts['showreadmore']) && $atts['showreadmore'] === 'true' ? 'true' : 'false';
    $order         = isset($atts['order']) && strtolower($atts['order']) === 'asc' ? 'ASC' : 'DESC';
    $orderby       = isset($atts['orderby']) ? sanitize_text_field($atts['orderby']) : 'date';

    $args = array(
        'post_type'          => 'post',
        'post_status'        => array('publish'),
        'order'              => $order,
        'orderby'            => $orderby,
        'posts_per_page'     => $posts_per_page,
        'ignore_sticky_posts'=> true,
    );
    if ( ! empty( $cat_slug ) ) {
        $args['tax_query'] = array( array(
            'taxonomy' => 'category',
            'terms'    => $cat_slug,
            'field'    => is_numeric($cat_slug) ? 'term_id' : 'slug',
        ) );
    }

    $query = new WP_Query( $args );
    $design_file = BDP_DIR . '/templates/grid/' . $design . '.php';
    ob_start();
    echo '<div class="bdp-post-grid-main bdp-' . esc_attr($design) . ' bdpgrid-' . esc_attr($grid) . ' bdp-clearfix">';
    if ( $query->have_posts() ) {
        while ( $query->have_posts() ) {
            $query->the_post();
            $count = 0; $cat_links = array(); $css_class = '';
            $post_featured_image = bdp_get_post_featured_image( $post->ID, $media_size );
            $post_link = bdp_get_post_link( $post->ID );
            $terms = get_the_terms( $post->ID, BDP_CAT );
            $tags = get_the_tag_list(' ',', ');
            $comments = get_comments_number( $post->ID );
            $reply = ($comments <= 1) ? __('Reply', 'bdp') : __('Replies', 'bdp');
            if( $terms ) { foreach($terms as $term) { $term_link = get_term_link($term); $cat_links[] = '<a href="'.esc_url($term_link).'">'.$term->name.'</a>'; } }
            $cate_name = join(' ', $cat_links);
            if ( file_exists($design_file) ) include( $design_file );
        }
    } else {
        echo '<p class="bdp-no-posts">' . esc_html__('No posts found.', 'blog-designer-pack') . '</p>';
    }
    wp_reset_postdata();
    echo '</div>';
    $html = ob_get_clean();
    wp_send_json_success( array( 'html' => $html ) );
}
Step 4: assets/js/bdp-public.js — Filter click handler
At the top of the file, after the existing initialization calls, add:
/* Category Filter */
jQuery(document).on('click', '.bdp-filter-btn', function() {
    var btn      = jQuery(this);
    var bar      = btn.closest('.bdp-filter-bar');
    var grid     = bar.next('.bdp-post-grid-main');
    var cat      = btn.data('cat');
    var conf     = bar.data('conf');

    bar.find('.bdp-filter-btn').removeClass('active');
    btn.addClass('active');
    grid.css('opacity', '0.5');

    jQuery.post( Wpbdp.ajaxurl, {
        action : 'bdp_filter_posts',
        nonce  : Wpbdp.filter_nonce,
        cat    : cat,
        conf   : conf
    }, function(res) {
        if ( res.success ) {
            grid.replaceWith( res.data.html );
        }
        bar.next('.bdp-post-grid-main').css('opacity', '1');
    });
});

Shortcode Usage After Implementation
Standard usage — no filter:
[bdp_post limit="12" grid="3" design="design-1"]
With category filter buttons:
[bdp_post limit="12" grid="3" design="design-1" show_filter="true" filter_cats="travel,food,lifestyle"]
Or by category ID:
[bdp_post limit="12" grid="3" show_filter="true" filter_cats="4,7,12"]
